MatchDay — App Icon
===================

Pitch sphere mark: a football and a pitch (equator, centre circle, centre spot)
in one geometric mark. White ball, black markings, solid near-black background.

Files
-----
matchday-icon-512.png   512 x 512  — master (iOS, App Store, general use)
matchday-icon-192.png   192 x 192  — Android launcher / PWA
matchday-icon.svg       vector source — scale to any size, edit colours freely

Colours
-------
Background   #0B0C14
Ball         #FFFFFF -> #E2E5F4 (radial)
Markings     #0B0C14

Notes
-----
- PNGs have a solid background (no transparency), ready for masking on iOS/Android.
- The mark sits inside the safe zone, so it survives circular & squircle masks.
- For other sizes (e.g. 1024 for App Store), re-export from the SVG at any resolution.
